# Retail Demand Forecasting

This project predicts retail product demand using Machine Learning.

## Objective
To forecast product order demand using historical retail data and time-based features.

## Algorithm Used
Random Forest Regressor

## Features Used
- Year
- Month
- Day
- Weekday
- Quarter
- Weekend
- Promotion Impact
- Previous Demand
- Rolling Mean 7

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Output
The model predicts future order demand to help businesses manage inventory, reduce overstocking, and avoid stockouts.
